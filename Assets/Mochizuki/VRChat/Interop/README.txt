# UdonRabbit - Interop v0.2.0-alpha.1

以下の機能を備えたワールド用 Udon / UdonSharp ギミック・ライブラリです。
主に開発者向けですが、利用者にもダウンロードが必要です。

・基本的な動作をするコンポーネント集
・なつねこらぼらとりーで配布しているパッケージの組み合わせを可能にします
・他の人もこのプロトコルに従うことで、なつねこらぼらとりーで配布しているパッケージとその他の製品とを組み合わせを可能にします


# 注意事項

不具合などがあるかもしれません。あらかじめご了承ください。

また、 VRChat SDK3 / Udon / UdonSharp の今後のアップデートによって不具合などが発生する可能性もあります。

なお、以下の環境において動作確認を行っております。
・VRChat Udon Spring 2021 Networking Beta
・VRCSDK3 WORLD Udon Networking Beta v4
・UdonSharp 0.19.6

# 開発者向け

UdonRabbit Interop を用いることで、開発者は以下の利益を得ることが出来ます。

・UdonSharp で型安全な状態で組み合わせ可能なコンポーネントが制作可能
・コンポーネント間でのパラメータの受け渡しが可能
・バリデーター機能で、コンポーネント間で組み合わせが不可能な場合のユーザーへの通知
